A colour variant of the [Surfn](https://github.com/erikdubois/surfn) icon theme. Ships only the **Surfn-Mint-Y-Aqua** theme and inherits the base **Surfn** icon set, so `surfn-icons-git` is required. Folder-colour icons are sourced from [linuxmint/mint-y-icons](https://github.com/linuxmint/mint-y-icons).

## Installation (Arch / Kiro — nemesis_repo)

```
sudo pacman -S surfn-mint-y-aqua-icons-git
```

`surfn-icons-git` (base Surfn) is pulled in automatically as a dependency.

## Manual

Copy the **Surfn-Mint-Y-Aqua** icon folder into `~/.icons` (or `~/.local/share/icons` on Plasma),
make sure the base `Surfn` theme is present, then select **Surfn Mint-Y-Aqua** in your appearance settings.
